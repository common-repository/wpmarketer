== Description ==

WPMarketer is a powerful suite of tools that extend the WordPress platform, enabling you to manage and track all of your online marketing and advertising efforts directly from your WordPress admin panel. Manage all of your affiliate links and conversions without having to use multiple tools to set up ads in posts and landing pages, manage your inventory and track results. Now you can manage your online marketing with greater control and ease of use than ever before.

= Plugin Features =

* Build a Performance Marketing category and link hierarchy using HTML text, HTML image and JavaScript &#8220;Link Types&#8221;
* You can easily add virtually any kind of link or Ad to your Wordpress posts and pages from Adsense to HTML or JavaScript Affiliate links/ads/banners, etc.
* Auto Link or insert our simple tags in your posts and pages; all links and ads automatically update whenever you edit a link
* Build your own &#8220;In Links&#8221;, which lets you auto link keywords on your posts and pages to earn money from sponsors ads, affiliate and performance marketing offers and more
* Affiliate Link masking/cloaking option with &#8220;pretty&#8221; internal URLs feature
* Follow/Nofollow option for text and image links
* Google Adwords conversion tracking
* Link/Ad click tracking

== Installation ==

1. Download & extract plugin using its default directory
1. Upload plugin directory to the &#8216;/wp-content/plugins/&#8217; directory. If you&#8217;re uploading it make sure to upload the top-level folder. Don&#8217;t just upload all the php files and put them in `/wp-content/plugins/`
1. Activate the plugin through the &#8216;Plugins&#8217; menu in WordPress
1. Visit your WPMarketer admin panel: WP-Admin - Settings - WPMarketer
1. That&#8217;s it! Start setting up your affiliate marketing links
